Train Control Commands

Movement Commands
speed <0-31> Set train speed (0=stop, 31=maximum)
forward Set direction to forward
reverse Set direction to reverse
toggle Toggle direction (forward ‚Üî reverse)
Sound Commands
horn Blow the train horn
bell on Turn bell on
bell off Turn bell off
Lighting Commands
lights on Turn train lights on
lights off Turn train lights off
Control Flow
wait <seconds> Pause execution (supports decimals, e.g., 0.5)
repeat <n> times Start a loop that repeats n times
end End a repeat loop
# comment Comments are ignored (must start line)

Example Scripts

Simple Demonstration: 
# Start from stop
speed 0
bell on
lights on
wait 2

# Accelerate forward
forward
speed 10
wait 3
speed 20
wait 3

# Horn and slow down
horn
wait 1
speed 10
wait 2
speed 0
bell off
lights off
Back and Forth Pattern: 
# Repeat pattern 3 times
repeat 3 times
    forward
    speed 15
    wait 5
    speed 0
    wait 1

    reverse
    speed 15
    wait 5
    speed 0
    wait 1
end

# Final horn
horn
wait 0.5
horn
Nested Loops: 
# Complex pattern with nested loops
repeat 2 times
    forward
    speed 20

    # Horn sequence
    repeat 3 times
        horn
        wait 0.5
    end

    wait 3
    speed 0
    wait 1
end
üí° Tips:
  • Use comments (#) to document your scripts
  • Scripts are validated before saving
  • Test scripts with "Run Now" before scheduling
  • One command per line
  • Indentation is optional but improves readability
  • All loops must have matching "end" statements
