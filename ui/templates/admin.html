Available Train Controls

Warning: Disabling controls will prevent users from accessing them. An admin password is required to change these settings.
Speed Control
Allow users to control train speed
Direction Control
Allow users to change train direction
Horn Control
Allow users to blow the train horn
Bell Control
Allow users to ring the train bell
Emergency Stop - All Users
Allow anyone in queue to trigger emergency stop (default: only current controller + admin)
